Esse é meu primeiro projeto JS publicado no github. Vou fazer um contador. Projeto aprendido no bootcamp realizado pela plataforma DIO. Usarei os modos mais antigos da escrita em JS com o método addEventListener e "onclick".
Será feito, para cada modo, 3 condições. 1 modelo é proposto no curso, sem esconder botões ou alterar cor. Outros 2 terão condicionais.
1 - O botão de diminuir será excluido quando o valor chegar a 0.
2 - O texto ficará em vermelho quando o valor for menor que 0.
